Enumerable Methods Cheatsheet

Jack Thatcher

Dev Bootcamp

Iteration

  • each { |x| } -- Yields all elements
  • each_with_index(*args) { |x, i| } -- Yields all elements with their indices
  • reverse_each { |x| } -- Yields all elements in reverse order.

Boolean

  • include?(value) -- Returns true if any element == value.
  • all? [{|x| }] -- Returns true if block never returns “false”.
  • any? [{|x| }] -- Returns true if block ever returns “true”. Default block { |x| x }.
  • none? [{ |x| }] -- Returns true if block never returns “true”. Default block { |x| x }

Sorting

  • sort [{ |a, b| }] -- Returns array sorted by <=> operator or by block.
  • sort_by { |x| } -- Returns array sorted by the return value of the block.
  • max [{ |a, b| }] -- Returns maximum element
  • min [{ |a, b| }] -- Returns minimum element

Filter

  • first() -- Returns the first element
  • take(n) -- Returns array ofo first n element
  • drop(n) -- Returns array of all elements after the first n
  • map { |x| } -- Returns an array with the results from the block

Dividing

  • chunk { |x| } -- Returns enumerator for consecutive chunks of elements with common block value
  • slice_before(pattern) -- Returns enumerator for consecutive chunks of elements
  • group_by { |x| } -- Returns a hash associating block return values to arrays of elements.
